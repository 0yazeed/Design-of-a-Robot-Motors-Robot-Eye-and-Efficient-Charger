# Controlling Robot Motors Using L298N



### Used equipments

* Arduino
* L298N Motor Driver
* 2 motors 
* **11.1 V & 2200mAH** Battery 



### Implementation Steps

At first i connect the arduino and L298N ports as follow: 
| Arduino     | L298N |
| ---      | ---       |
| 13       | Enaple A  |
| 12       | IN1       |
| 11       | IN2       |
| 10       | IN3       |
| 9        | IN4       |
| 8        | Enaple b  |
